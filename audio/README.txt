Nhạc nền cho web game
=====================

Để có nhạc nền khi chơi, đặt file nhạc (MP3) vào thư mục này với tên:

  bg-music.mp3

Gợi ý tải nhạc miễn phí, phù hợp trẻ em:
- Pixabay Music: https://pixabay.com/music/
- YouTube Audio Library (tải file rồi đổi tên thành bg-music.mp3)

Sau khi có file bg-music.mp3, mở lại trang game và bấm "Bật nhạc nền" (hoặc nhạc sẽ tự bật nếu đã bật trước đó).
