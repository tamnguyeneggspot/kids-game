# Kế hoạch Web Game cho trẻ em

## 1. Ý tưởng game phù hợp

### Bảng chữ cái (Alphabet)
- **Ghép chữ với hình**: chọn hình tương ứng với chữ cái (A = Apple, B = Ball...).
- **Nghe và chọn**: phát âm chữ cái, trẻ chọn đúng ô chữ.
- **Sắp xếp A–Z**: kéo thả các chữ vào đúng thứ tự.
- **Tô màu chữ**: tô theo nét, nhận diện chữ in hoa / thường.

### Số đếm (Counting)
- **Đếm đồ vật**: đếm số lượng hình (sao, quả, bóng) rồi chọn số.
- **Nối số với số lượng**: nối số 3 với nhóm có 3 con vật.
- **Điền số còn thiếu**: 1, 2, _, 4 → điền 3.
- **So sánh nhiều/ít**: chọn bên có nhiều hơn hoặc ít hơn.

### Kết hợp (Alphabet + Numbers)
- **Trang chủ mini**: nhiều game nhỏ (chữ cái, số đếm, màu sắc) chọn bằng icon.

## 2. Công nghệ gợi ý

- **Frontend**: HTML + CSS + JavaScript thuần (dễ deploy, chạy offline).
- **Âm thanh**: Web Speech API (đọc chữ/số) hoặc file audio có sẵn.
- **Giao diện**: lớn, rõ, nhiều màu; nút bấm to; có âm báo khi đúng/sai.

## 3. Cấu trúc thư mục đề xuất

  WebGame/
  ├── index.html          # Trang chủ chọn game
  ├── css/
  │   └── style.css
  ├── js/
  │   ├── app.js          # Logic chung

  │   └── counting.js     # Game số đếm
  └── assets/
      ├── images/         # Hình minh họa (A=apple, 1=one ball...)
      └── sounds/         # Âm thanh (đúng/sai, đọc chữ/số)

## 4. Bước làm đầu tiên

1. Làm **trang chủ** với 2 nút: "Bảng chữ cái" và "Số đếm".
2. Làm **1 game đơn giản** trước (ví dụ: Đếm đồ vật – hiện N hình, chọn số).
3. Sau đó thêm game chữ cái (ví dụ: Ghép chữ với hình).

*(Đã xong: trang chủ, Số đếm Nhìn/Nghe/Test, Bảng chữ cái Nhìn/Nghe, Màu sắc Nhìn/Nghe.)*

## 5. Bước làm tiếp theo (ưu tiên)

### 5.1. Ghép chữ với hình (Bảng chữ cái)
- Hiện một chữ cái (hoặc hình), bé chọn hình (hoặc chữ) tương ứng: A = Apple, B = Ball...
- Có thể thêm tab/mode mới trong alphabet.html hoặc trang riêng.
- Dùng emoji hoặc ảnh trong img/ (nếu có) cho hình minh họa.

### 5.2. So sánh nhiều/ít (Số đếm)
- Hiện 2 nhóm đồ vật (trái vs phải), câu hỏi: "Bên nào nhiều hơn?" hoặc "Bên nào ít hơn?".
- Bé chọn bên trái hoặc bên phải.
- Logic: so sánh 2 số, không cần thêm asset phức tạp.

### 5.3. Điền số còn thiếu (Số đếm)
- Hiện dãy số có ô trống: ví dụ 1, 2, _, 4.
- Bé chọn số đúng (3) trong 3–4 đáp án.
- Tăng độ khó nhẹ so với đếm đồ vật.

### 5.4. Chế độ Test cho Màu sắc
- Giống Số đếm: thêm tab "Test" trong color.html.
- Câu hỏi: "Màu này gọi là gì?" → 4 đáp án (tên màu), bé chọn.
- Tái dùng pattern từ counting.js (chế độ Test).

### 5.5. (Tùy chọn) Các ý tưởng khác từ mục 1
- **Sắp xếp A–Z**: kéo thả chữ vào đúng thứ tự (Alphabet).
- **Nối số với số lượng**: nối số 3 với nhóm có 3 con vật (Counting).
- **Tô màu chữ**: tô theo nét (Alphabet) — cần canvas/ SVG, phức tạp hơn.
